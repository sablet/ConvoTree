# クエリパラメータ抽出

以下のユーザークエリから、検索に必要なパラメータを抽出してください。

## ユーザークエリ

{{QUERY}}

## 抽出すべきパラメータ

1. **期間（period_days）**: 「ここ1週間」→7、「ここ数日」→3、「ここ1ヶ月」→30。期間の指定がない場合はnull。
2. **トピック（topic）**:
   - クエリの主題をトピックとして抽出してください。
   - 例:
     - 「開発ツールについて何をやっていたか」→ "開発ツール"
     - 「旅行の目的を明確にしたい」→ "旅行の目的"
     - 「アルゴトレードの進捗は？」→ "アルゴトレード"
   - **重要**: クエリに何らかの主題が含まれている場合、必ずtopicに抽出してください。
   - 主題が全く不明な場合のみnull。
3. **ステータスフィルタ（status_filter）**:
   - 常に全ステータスを対象とする: ["todo", "idea", "doing", "done"]

## 出力形式

JSON形式で出力してください。他の説明は不要です。

```json
{
  "period_days": 7,
  "topic": "開発ツール",
  "status_filter": ["todo", "idea", "doing", "done"]
}
```
