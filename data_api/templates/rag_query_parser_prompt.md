# クエリパラメータ抽出

以下のユーザークエリから、検索に必要なパラメータを抽出してください。

## ユーザークエリ

{{QUERY}}

## 抽出すべきパラメータ

1. **期間（period_days）**: 「ここ1週間」→7、「ここ数日」→3、「ここ1ヶ月」→30。指定がない場合はnull。
2. **トピック（topic）**: 検索対象のトピック（例: 「開発ツール」「アルゴトレード」）。指定がない場合はnull。
3. **ステータスフィルタ（status_filter）**:
   <!-- 一時的に無効化: goal network にステータス網羅が不足しているため、推定が機能していない -->
   <!-- - 「何をやっていたか」「やったこと」→ ["doing", "done"] -->
   <!-- - 「次にやるべき」「これからやること」→ ["todo", "idea"] -->
   - 常に全ステータスを対象とする: ["todo", "idea", "doing", "done"]

## 出力形式

JSON形式で出力してください。他の説明は不要です。

```json
{
  "period_days": 7,
  "topic": "開発ツール",
  "status_filter": ["todo", "idea", "doing", "done"]
}
```
